# Agent

**N/A — no agent framework.** Anaj Bahi is a deterministic, offline-first data/ledger PWA. It has **no LLM, no agent orchestration, no prompts, and no model provider**. There is no agent graph to specify. All logic is plain TypeScript: a pure calculation module (`lib/calc`) and an IndexedDB repository (`lib/db`). See [architecture.md](architecture.md) and [data.md](data.md).
